# AutoMarket 
Автомаркет - это сервис по поиску и продаже автомобилей, написанный на .NET 8.0, использующий стек:

* ASP.NET Core 8.0
* Entity Framework Core 8.0
* PostgreSQL
* Docker
* RabbitMQ
* Swagger
* JWT
